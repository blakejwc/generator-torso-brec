# generator-torso-brec
Yeoman generator for a project that uses torso and brec

> npm install --global generator-torso-brec
> yo torso-brec

returns:
  
  ├─ .gitignore  
  ├─ package.json  
  ├─ gulpfile.js  
  ├─ dist  
  └─ app  
     ├─ main.js  
     ├─ app.scss  
     ├─ index.html  
     └─ home  
        ├─ homeView.js  
        └─ home-template.hbs
